# Purpose

This document defines how every design decision inside NorthStar is made.

It translates the Brand DNA and Design Bible into practical design execution.

Whenever uncertainty exists,

this document takes precedence over personal preference.

---

# Primary Design Inputs

Every design task should use the following priority:

1. NorthStar Constitution
2. NorthStar Brand DNA
3. Design Bible
4. Product Bible
5. Product Specifications

Never skip this order.

---

# Design Decision Framework

Before creating any new screen or component,

always ask:

- Does this improve understanding?
- Does this reduce cognitive load?
- Does this strengthen trust?
- Does this match Brand DNA?
- Is it consistent with the Design Bible?

If any answer is "No",

reconsider the design.

---

# Design Priorities

Always prioritize:

1. Trust
2. Understanding
3. Clarity
4. Consistency
5. Accessibility
6. Beauty

Visual beauty should never reduce usability.

---

# Design Constraints

Never introduce UI elements simply because they are fashionable.

Avoid unnecessary:

- animations
- gradients
- shadows
- decorative illustrations
- visual noise

Every visual element must have a purpose.

---

# Component Rules

Every component should:

- solve one problem
- be reusable
- follow Design Tokens
- support accessibility
- behave consistently

Never duplicate components unnecessarily.

---

# AI Design Rules

AI is part of the interface.

AI should always:

- explain
- educate
- support
- clarify

AI should never:

- interrupt
- pressure
- manipulate
- distract

---

# Review Checklist

Before approving any screen,

verify:

✓ Brand DNA respected
✓ Design Bible followed
✓ Product requirements satisfied
✓ Accessibility maintained
✓ Visual consistency preserved
✓ Information hierarchy clear
✓ Cognitive load minimized

---

# Definition of Done

A design task is complete only when:

- users understand it quickly
- AI integrates naturally
- the screen feels calm
- visual consistency is maintained
- no unnecessary elements remain

---

# Success

NorthStar should become recognizable not because it looks different,

but because it feels different.

Understanding is the visual identity.
