# AI Product OS Constitution

Version: 1.0
Status: Draft
Owner: Founder

---

# Purpose

This repository is the operating system used to build products with specialized roles. It exists so that product knowledge, decisions, and process outlive any individual tool used to execute them.

---

# Vision

A durable foundation on which any AI-assisted product can be built, using the same roles, ownership model, and workflow — regardless of which tools are used to execute the work at any given time.

---

# Mission

Provide the permanent structure — roles, responsibilities, workflow, and repository rules — that every product built on this OS inherits from day one.

---

# Core Principles

- The repository is the single source of truth. Documentation overrides assumption or memory.
- Roles are permanent. Tools are replaceable.
- Every artifact has exactly one owner.
- One role cannot silently modify another role's approved work.
- Documentation precedes implementation.
- Simplicity is preferred over complexity whenever both deliver equal value.

---

# Decision Hierarchy

Whenever guidance conflicts, the following order governs:

1. AI Product OS Constitution
2. Organization Charter
3. Operating Model
4. Workflow Overview
5. Repository Standard
6. Individual product specifications (e.g. NorthStar and future reference implementations)

Higher levels may never be silently overridden by lower ones.

---

# Non-Negotiable Rules

- The repository is the source of truth, not any individual's memory or assumption.
- Every artifact has exactly one owner at all times.
- No role modifies another role's approved work without that role's involvement.
- Roles are permanent; the tools fulfilling them may change without changing this Constitution.
- This Constitution is not silently reinterpreted — changes require an explicit, recorded amendment.

---

# Definition of Success

This OS succeeds when any product can be built by following its documented roles, ownership model, and workflow alone — reconstructable from repository history, independent of which tools were used to execute the work.
